About This Project

Student Cost Forecast is a web-based tool designed to help Nigerian undergraduates estimate their annual cost of living based on real student data and predictive modeling.

It uses a trained machine learning model (Linear Regression) that considers factors like gender, lifestyle, accommodation type, income sources, and financial aid to produce accurate predictions.

This platform is ideal for students, parents, education planners, and school administrators who want a data-driven understanding of student financial needs.

Built with ❤️ using Flask, Bootstrap, Pandas, and Scikit-learn.
